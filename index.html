Blockchain 2.0

Smart contracts

Blocks in the chain begin to be used for more than a transaction record -- beyond just an exchange.

Because you can hold any data in a block, it can also hold code to execute or enforce contracts, and it can do so automatically.

Like a vending machine: you put in your money, make a selection, and you get the candy bar. No negotiation, no variation.

Adds business logic to a transaction. So bitcoin is just the transaction, but the next generation of blockchains introduce the idea of transactions plus conditions.

Ethereum

Decentralized Applications

Ethereum builds a new blockchain, introduced in Jan 2014.

From their website:

"Ethereum is a decentralized platform that runs smart contracts: applications that run exactly as programmed without any possibility of downtime, censorship, fraud or third-party interference."

Other blockchains start appearing: Ripple, Hyperledger. Some are public and open source; others are private. Their primary intended uses differ.

Possibilities for decentralized applications include the exchange of currency but other things as well:

  • file storage
  • ridesharing
  • social messaging
  • buying/selling physical items
